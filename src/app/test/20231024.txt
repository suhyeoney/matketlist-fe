- 기간 : 2023.01.02 ~ 2023.03.10
- 사용 기술 : React (React Bootstrap, Recoil, React Query), Typescript
- 담당 업무 :
1. Front-end 배포를 위한 AWS CodePipeline-S3-Cloudfront based CICD 프로세스 구축
* S3 > index.html 버킷 파일 업로드
* CodeBuild > buildspec.yaml 파일 작성
* CloudFront > S3 버킷 파일과 같은 정적 컨텐츠 배포 (CDN)
2. 대한민국 정부에서 수주하는 모든 계약건 raw 데이터를 기반으로 각종 검색 및 현황 기능을 제공하는 화면 개발

- 주요 실적 (역량) :
1. 프로젝트 아키텍팅부터 공통, 검색기능, 필터기능, 지역별 계약현황 차트에 이르기까지 요구사항에 대한 모든 내역들을 함수형 컴포넌트방식으로 개발 완료,
2. React-based 신기술 적용능력, AWS CICD 프로세스 구축능력, 웹 디자이너와의 협업 및 일정관리를 포함한 커뮤니케이션능력을 해당 과제에서 발휘했습니다.
- 웹 디자이너 협업 > React Bootstrap + SCSS 퍼블리싱
